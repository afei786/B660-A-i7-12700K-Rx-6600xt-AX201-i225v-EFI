# ROG-STRIX-B660-A-GAMING-WIFI-i7-12700K-Rx-6600xt-EFI

## 配置

    - CPU i7 12700k
    - GPU AMD RX 6600xt
    - 主板 华硕 b660-a strix gaming wifi d4 
    - 有线网卡型号 intel i225-v 
    - 无线网卡型号 intel ax201
    - 内存 16g*2
    - 硬盘 500G + 1T

## 系统

    - windows 11 --> 1T 
    - MacOS 13.5.1 (安装的是12.5镜像，后升级到13.5.1) --> 500G
    - OpenCore 0.9.1

## BIOS设置

    - 关闭VT-d
    - 关闭CFG Lock
    - 关闭CSM
    - 关闭安全启动
    - 关闭快速启动
